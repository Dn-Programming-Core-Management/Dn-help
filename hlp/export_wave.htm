Export to wave

This feature allows modules to be exported as wave files. The samplerate, filtering and volume output follow the module's sound configuration.

Song length

The song can be exported either in loop counts or in a specified length of time.

Song

Selects the song to be exported.

Channels

Selects the channels to be audible in export.

Seperate channel export

If enabled, exports a single isolated wave file for each channel selected in the Channels list.
